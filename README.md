# Capstone Project Hidden Markov Model and Bidirectional Long short term memory

The paper I chose was a tutorial on Hidden Markov Model.

The paper talks about speech signals and how they are difficult to classify and that there is a need for a stochastic model that would classify it better.

The repository contains two main file.

VaidehiKaliyaperumal_CapstoneProject_HMM

This file is the replication of the main paper with the code for HMM written from scratch. The file contains all the necessary information to run the file. It is a notebook with the results present in it.

VaidehiKaliyaperumal_CapstoneProject_BiLSTM

The file is the code I had written for Bidirectional LSTM,to compare with the results of the HMM.It is a notebook and after all dependencies are installed run cell by cell.

It could be seen that HMM has a accurcay of 0.95 compared to 0.87 of BiLSTM proving the claim of the author that HMM works well for speech dataset.

Paper source: https://ieeexplore.ieee.org/document/18626

Dataset source: https://archive.ics.uci.edu/ml/datasets/japanese+vowels
